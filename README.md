# Authentication-module

## Importante:
Favor crear .env en el folder Database, que contenga:

```env
POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_PORT=
```

Favor crear .env en el folder Authentication que contenga:

```env
POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=
POSTGRES_PORT=
POSTGRES_IP=
SPRING_PORT=
```

Buildear por fuera el paquete con:
```bash
    mvn clean package -DskipTests
```
